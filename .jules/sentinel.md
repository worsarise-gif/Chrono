## 2024-05-24 - Hardcoded API Keys as Fallback Mechanism
**Vulnerability:** Hardcoded Cloudflare API credentials (account ID and token) were found in source code files `src/app/api/cloudflare-chat/route.ts` and `src/app/api/generate-image/route.ts` acting as a fallback when environment variables were not provided.
**Learning:** Hardcoding credentials, even as a fallback, exposes sensitive keys directly in the codebase which is a critical security vulnerability, as they could be leaked if the repository is exposed or shared. The system was designed to have a failsafe when environment credentials were not configured, but compromised security to do so.
**Prevention:** Never commit API keys or secrets into source control. Always rely on robust secret management (like environment variables). If credentials are missing, fail securely with an appropriate error (e.g., throwing an error or returning a 500 response indicating missing configuration) rather than injecting hardcoded fallback credentials.
